# nym 시작하기 (Korean Guide)

## 프로젝트 완성! 🎉

**nym** - Monad 기반 Web3 앱 빌더가 완성되었습니다!

## ✅ 구현된 기능

### 1. 이중 AI 에이전트 시스템
- **빌더 에이전트**: 좌측 사이드바에 항상 표시, 코딩 도움
- **질문 에이전트**: 우측 하단 팝업, 기획 및 디자인 도움

### 2. 완전한 웹 기반 IDE
- Monaco 에디터 (VSCode와 동일한 경험)
- 파일 탐색기
- 실시간 라이브 프리뷰
- 코드/프리뷰 전환

### 3. 워크스페이스 관리
- 여러 워크스페이스 생성
- 워크스페이스별 파일 저장
- 지갑 주소와 연동

### 4. Web3 통합
- MetaMask 연결
- Monad Mainnet 준비 완료

### 5. 저장 및 내보내기
- ZIP 파일로 다운로드
- 데이터베이스 자동 저장

## 🚀 빠른 시작

### 1. 필수 요구사항
- Node.js 18 이상
- PostgreSQL 14 이상
- OpenAI API 키

### 2. 설치

```bash
# Windows에서:
.\setup.ps1

# Mac/Linux에서:
chmod +x setup.sh
./setup.sh
```

### 3. 환경 변수 설정

**backend/.env 파일 생성:**
```env
DATABASE_URL=postgresql://사용자명:비밀번호@localhost:5432/nym
OPENAI_API_KEY=sk-여기에-OpenAI-키-입력
PORT=3001
NODE_ENV=development
FRONTEND_URL=http://localhost:3000
```

**frontend/.env.local 파일 생성:**
```env
NEXT_PUBLIC_API_URL=http://localhost:3001
NEXT_PUBLIC_MONAD_RPC_URL=https://rpc.monad.xyz
```

### 4. 데이터베이스 생성

```sql
CREATE DATABASE nym;
```

테이블은 첫 실행 시 자동으로 생성됩니다.

### 5. 개발 서버 시작

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속!

## 💡 사용 방법

### 첫 번째 앱 만들기

1. **워크스페이스 생성**
   - 워크스페이스 선택기에서 "+" 버튼 클릭
   - 이름 입력 (예: "첫 번째 앱")
   - "Create" 클릭

2. **빌더 에이전트와 대화**
   - 좌측 채팅창에 입력: "Create a landing page with a hero section"
   - AI가 코드를 생성합니다!

3. **라이브 프리뷰 보기**
   - "Preview" 탭을 클릭하여 실시간으로 확인
   - "Code" 탭으로 전환하여 직접 편집

4. **질문 에이전트 사용**
   - 우측 하단 "Need Help?" 버튼 클릭
   - 기획이나 디자인에 대해 질문

5. **작업 저장**
   - 상단 "Save" 버튼으로 ZIP 다운로드
   - 모든 내용은 데이터베이스에 자동 저장

## 🔍 문제 해결

### "Database connection failed"
- PostgreSQL 실행 확인
- DATABASE_URL 확인
- 데이터베이스 존재 확인

### "OpenAI API error"
- API 키 확인
- 크레딧 확인
- OpenAI 상태 페이지 확인

### 포트 충돌
- backend/.env의 PORT 변경
- frontend/.env.local의 NEXT_PUBLIC_API_URL 업데이트
